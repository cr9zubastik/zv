# ZV - Мобильное антивирусное приложение

ZV - это мобильное антивирусное приложение для Android, разработанное с использованием современных технологий для обеспечения безопасности ваших файлов и URL-адресов.

## Описание

ZV позволяет сканировать файлы и URL-адреса на наличие угроз, используя несколько антивирусных движков через внешние API. Приложение предоставляет удобный интерфейс для проверки безопасности и ведения истории обнаруженных угроз.

## Основные возможности

- Аутентификация пользователей
  - Регистрация новых пользователей
  - Вход в систему
  - Гостевой режим для быстрого доступа

- Сканирование файлов
  - Проверка файлов на наличие угроз
  - Вычисление SHA-256 хеша файлов
  - Анализ через несколько антивирусных движков

- Сканирование URL
  - Проверка веб-адресов на безопасность
  - Определение вредоносных ссылок

- История угроз
  - Сохранение всех результатов сканирования
  - Просмотр истории обнаруженных угроз
  - Детальная информация о каждой угрозе

- Информация об устройстве
  - Просмотр сведений о вашем устройстве

- Настройки
  - Управление профилем
  - Выход из аккаунта

## Технологии

### Основной стек
- Kotlin - язык программирования
- Jetpack Compose - современный UI фреймворк
- Material Design 3 - дизайн-система

### Аутентификация
- Firebase Authentication - система аутентификации и авторизации

### API интеграции
- OTX API (AlienVault Open Threat Exchange) - проверка файлов и URL на угрозы
- VirusTotal API - многодвижковое сканирование файлов и URL

### Локальное хранилище
- Room Database - база данных для хранения истории угроз

### Сетевые библиотеки
- Retrofit - HTTP клиент для работы с API
- OkHttp - HTTP клиент с логированием запросов
- Gson - сериализация/десериализация JSON

### Навигация
- Navigation Compose - навигация между экранами
- ViewModel - управление состоянием экранов

### Разрешения
- Accompanist Permissions - управление разрешениями Android

## Требования

- Android 8.1 (API 27) или выше
- Интернет-соединение для работы с API
- Разрешения на чтение файлов (для сканирования файлов)

## Установка

1. Клонируйте репозиторий:
```bash
git clone https://github.com/cr9zubastik/zv
cd zv
```

2. Откройте проект в Android Studio

3. Соберите проект:
```bash
./gradlew build
```

Примечание: Firebase Console и API ключи для OTX и VirusTotal уже настроены и внедрены в проект.

## Основные экраны

- LoginScreen - экран входа в систему
- RegisterScreen - экран регистрации
- ScannerScreen - экран сканирования файлов и URL
- ThreatHistoryScreen - экран истории обнаруженных угроз
- DeviceInfoScreen - экран информации об устройстве
- SettingsScreen - экран настроек

## Безопасность

- Все файлы сканируются через вычисление SHA-256 хеша
- Используются проверенные антивирусные API (OTX, VirusTotal)
- Результаты сканирования сохраняются локально в зашифрованной базе данных
- Аутентификация через Firebase обеспечивает безопасный доступ

## Уровни угроз

Приложение определяет следующие уровни угроз:
- SAFE - безопасно
- LOW - низкий уровень угрозы
- MEDIUM - средний уровень угрозы
- HIGH - высокий уровень угрозы
